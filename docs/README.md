# Forge Senior Project Portfolio

This is a static Senior Project Portfolio website for Forge.

## Included

- `index.html` - Main portfolio page
- `styles.css` - Styling for the portfolio
- `assets/` - Forge screenshots and SVG design diagrams

## Current content

This version includes:
- Project overview and background
- Main Forge features
- Application flow, component, and data/calculation diagrams
- Screenshots for dashboard, profile, metrics/macros, workout tracker, workout history, cardio tracker, and cardio history
- Code snippets for calculation logic, macro targets, progressive overload feedback, and JSON storage
- Basic run instructions
- Supporting artifact link placeholders

## GitHub Pages quick path

1. Add these files to a GitHub repository.
2. Go to Settings > Pages.
3. Choose the branch and root folder.
4. Save and wait for GitHub to generate the public site URL.
